# UIkit

Статическая среда для:

- примитивов
- блоков
- визуальных сборок
- ручной проверки HTML/CSS до переноса в `frontend/`

Локальный запуск:

```bash
python3 -m http.server 4173 -d UIkit
```

Точки входа:

- `http://localhost:4173/index.html`
- `http://localhost:4173/primitives.html`
- `http://localhost:4173/blocks.html`
